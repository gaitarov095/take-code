# ⚡ TakeCode

> **TakeCode** is a modern, blazing-fast, and elegant Snippet Management Hub built for developers to organize, verify, and instantly reuse code patterns. Stop reinventing the wheel, skip the boilerplate, and focus on delivering core features.

---

## ✨ Features

*   🌐 **Multi-language by Default:** Seamlessly discover and store snippets across frontend, backend, infrastructure, and systems code—without being forced into language silos.
*   ✅ **Verified Patterns:** Human-reviewed code highlights dependencies, tricky edge cases, and safe defaults.
*   🚀 **Copy in One Tap:** Intuitive UI elements, formatted code outputs, and clean metadata make reuse feel instantaneous.
*   📊 **Community Signal over Vanity Metrics:** Upvotes, bookmarks, and implementation notes help the best, battle-tested solutions rise to the top before you copy them to production.

---

## 🛠 Tech Stack

Built on top of cutting-edge, efficient tools:

*   **Frontend:** React + TypeScript + Tailwind CSS
*   **Design & UI:** Bento Grid layout, Catppuccin / Slate deep dark theme
*   **Syntax Highlighting:** Shiki (`codeToHtml`)
*   **Build Tool:** Vite
*   **Runtime & Package Manager:** Bun

---

## 🚀 Getting Started

### Prerequisites

To spin up the project locally, make sure you have [Bun](https://bun.sh/) (or Node.js / npm) installed.

### Local Setup
1. Clone the repository:
   ```bash
   git clone [https://github.com/gaitarov095/takecode.git](https://github.com/gaitarov095/takecode.git)
   cd takecode
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Spin up the development server:
   ```bash
   bun run dev
   ```

4. Open http://localhost:5173 in your favorite browser.
